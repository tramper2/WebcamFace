# User Flow: 가상 피팅 프로세스



1. **서비스 접속**: 사용자가 웹 사이트에 접속.
2. **권한 요청**: 웹캠 사용 권한 팝업 -> 승인 시 비디오 시작.
3. **얼굴 스캔**: 시스템이 얼굴을 감지할 때까지 "얼굴을 화면에 맞춰주세요" 메시지 표시.
4. **안경 선택**: 하단 카탈로그에서 원하는 안경 디자인 클릭.
5. **실시간 피팅**: 선택한 안경이 얼굴 위에 렌더링됨. 얼굴을 움직이면 안경도 함께 따라 움직임.
6. **인터랙션**: 
    - 안경 크기 미세 조정 (슬라이더 활용)
    - 다른 안경으로 교체
7. **결과 저장**: '촬영' 버튼 클릭 시 캔버스 이미지를 PNG로 다운로드.
