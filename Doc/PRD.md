# PRD: Web-based AI Eyewear Fitting Service

## 1. 개요
본 프로젝트는 웹캠을 통해 사용자의 얼굴을 실시간으로 인식하고, 선택한 안경 모델을 얼굴 위에 가상으로 착용(Virtual Try-on)해주는 웹 서비스이다.

## 2. 목적
- 사용자에게 안경 구매 전 가상 피팅 경험 제공
- 가볍고 빠른 웹 기반의 실시간 컴퓨터 비전 기술 구현 및 포트폴리오 활용

## 3. 주요 사용자 (User Personas)
- 온라인 안경 쇼핑몰 이용자
- 웹 기반 AR 기술에 관심 있는 개발자

## 4. 핵심 기능 (Core Features)
- **실시간 웹캠 스트리밍**: 브라우저 권한 획득 및 영상 출력.
- **얼굴 인식 및 랜드마크 추적**: 68개 이상의 안경 장착용 좌표 추출.
- **가상 안경 렌더링**: 얼굴 크기, 각도, 위치에 따른 안경 이미지 오버레이.
- **안경 카탈로그**: 다양한 디자인의 안경 선택 기능.
- **스크린샷**: 피팅된 모습을 이미지 파일로 저장.

## 5. 비기능적 요구사항
- 초당 24프레임(FPS) 이상의 부드러운 렌더링.
- 모바일 브라우저(Safari, Chrome) 대응.
- HTTPS 환경에서의 보안 정책 준수.
